# Test Generation Skill
# 测试生成技能 - 自动生成测试用例

## 用途
分析代码并自动生成测试用例，确保覆盖关键路径。

## 触发条件
- 新功能实现完成
- 测试覆盖率 < 80%
- 用户主动请求

## 使用方式

### 方式 1: 直接调用
```
请为以下函数生成测试：
[粘贴函数代码]
```

### 方式 2: 指定覆盖率目标
```
请为 [模块名] 生成测试，目标覆盖率 90%
```

## 输出格式
- 生成的测试代码
- 测试覆盖分析
- 未覆盖路径说明

## 安装
```bash
./install.sh test-generation codex    # 安装到 Codex
./install.sh test-generation cursor   # 安装到 Cursor
```
