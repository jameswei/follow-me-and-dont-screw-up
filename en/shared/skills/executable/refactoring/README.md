# Refactoring Skill
# 重构技能 - 识别代码坏味道并提供改进方案

## 用途
识别代码中的坏味道，提供安全的重构方案。

## 触发条件
- 检测到代码坏味道
- 代码复杂度超标
- 用户主动请求优化

## 使用方式

### 方式 1: 直接调用
```
请重构以下代码：
[粘贴代码]
```

### 方式 2: 指定重构目标
```
请提取以下代码中的重复逻辑：
[粘贴代码]
```

## 输出格式
- 识别出的坏味道
- 重构方案（多种选择）
- 重构后的代码
- 风险评估

## 安装
```bash
./install.sh refactoring codex    # 安装到 Codex
./install.sh refactoring cursor   # 安装到 Cursor
```
