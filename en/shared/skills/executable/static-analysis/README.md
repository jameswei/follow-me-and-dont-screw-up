# Static Analysis Skill
# 静态分析技能 - 自动化代码质量检查

## 用途
执行静态分析工具，捕获输出，自动修复问题，直到满足所有质量要求。

## 触发条件
- 代码保存后
- 提交前
- CI/CD 流程中
- 用户主动请求

## 使用方式

### 方式 1: 完整检查
```
请对当前代码执行完整静态分析
```

### 方式 2: 特定检查
```
请检查类型错误
```

### 方式 3: 自动修复
```
请修复所有自动可修复的问题
```

## 输出格式
- 问题列表（按严重程度排序）
- 自动修复建议
- 修复后的代码
- 验证结果

## 安装
```bash
./install.sh static-analysis codex    # 安装到 Codex
./install.sh static-analysis cursor   # 安装到 Cursor
```
