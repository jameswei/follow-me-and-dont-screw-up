# Code Review Skill
# 代码审查技能 - 系统性检查代码质量

## 用途
对代码变更进行系统性审查，识别问题并提供改进建议。

## 触发条件
- 文件变更 > 50 行
- 提交前审查
- 用户主动请求

## 使用方式

### 方式 1: 直接调用
```
请对以下代码进行审查：
[粘贴代码或 diff]
```

### 方式 2: 自动触发（在 Agent 配置中启用）
当检测到代码变更时自动执行。

## 输出格式
审查报告包含：
- 🔴 阻塞问题（必须修复）
- 🟡 高优先级问题（建议修复）
- 🟢 建议（可选改进）

## 审查维度
1. 正确性 - 逻辑错误、边界条件、并发问题
2. 可读性 - 命名、注释、复杂度
3. 可维护性 - 耦合度、重复、测试覆盖
4. 性能 - 算法复杂度、资源使用
5. 安全 - 注入、越界、敏感数据
6. 规范 - 风格、最佳实践

## 安装
```bash
./install.sh code-review codex    # 安装到 Codex
./install.sh code-review cursor   # 安装到 Cursor
```
