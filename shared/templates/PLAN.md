# 上下文恢复指南
# =============================================================================

## 如果会话中断，从此处恢复

1. **读取本文件**: 了解当前任务状态
2. **检查最后更新**: 确认信息时效性
3. **验证已完成任务**: 快速检查产出物是否存在
4. **确认当前任务**: 明确接下来要做什么
5. **询问用户**: "根据 PLAN.md，当前任务是 [X]，是否继续？"

## 会话恢复检查清单

- [ ] 已读取 PLAN.md
- [ ] 已确认当前任务状态
- [ ] 已检查依赖任务是否完成
- [ ] 已验证环境/代码状态
- [ ] 已向用户确认继续方向

# =============================================================================
# 使用指南
# =============================================================================

## 对于 Agent

1. **每次会话开始时**: 读取本文件，确认当前任务
2. **任务开始前**: 更新状态为 `🔄`，记录开始时间
3. **任务完成后**: 更新状态为 `✅`，记录实际工时，验收标准打勾
4. **遇到阻塞**: 更新状态为 `🚫`，记录原因和解除条件
5. **计划变更**: 在"变更日志"中记录，说明原因

## 对于用户

1. **审阅计划**: 确认任务分解合理、依赖正确
2. **调整优先级**: 根据业务需要调整 P0/P1/P2/P3
3. **跟踪进度**: 通过本文件了解项目状态
4. **批准变更**: 重大变更需要用户确认
